Basic Grammar of Python

发表于 2017-05-13 | 分类于 Programming Languages |

这段时间把《简明Python教程》学完并整理了Python基本语法的思维导图。
思维导图源文件下载

一些语法差异

引号

Python中不同引号之间互相兼容，不需要转义，但是单引号和三引号不能连在一起，否则会报错

# Right way
print '''The sample string was written by "Ryan"'''

如果要连在一起必须转义

# Don't do this
print '''The sample string was written by 'Ryan''''

# Do this
print '''The sample string was written by 'Ryan\''''

同引号之间不兼容，因为解释器分不清哪一段才是字符串。
另外，注意，双引号是一个符号，而三引号是三个单引号。

单引号或双引号的行末的一个反斜杠” \ “表示下一行继续，而不是新的一行，否则会报错，解释器会认为上下两行没有联系，而无法解释下一行的内容。使用三引号可以避免这种情况，因此通常单行内容比较长需要换行时可以用三引号。

print 'The first part of the line,\
the second part of the line but not a new line.'

List赋值

List赋值，应该采用切片操作符 “ : “

myList = fruit[:]

如果仅仅使用myList = fruit得到的是一个对fruit的引用，两个变量指向同一内存地址。

控制流

循环后可加else。总是会在循环结束后被执行一次，除非遇到break。
换句话说，循环后的else通常被用在，break后不需要执行，而其他时候下需要执行。

异常

与C，Java，C#不同的是，Python处理异常的语句是try...except...finally，而不是try...catch...finally

Python示例：

import sys

class ShortInputException(Exception):
    '''If the length of string is less than the given number,
    this exception will be raised.'''

    def __init__(self, length, atLeast):
        Exception.__init__(self)
        self.length = length
        self.atLeast = atLeast

atLeast = 3

try:
    string = raw_input("Enter a string here=>")
    if len(string) < atLeast:
        raise ShortInputException(len(string), atLeast)
except EOFError:
    print 'Why did you do an EOFError on me?'
except ShortInputException, x:
    print 'The length of string you entered was %d,\
    but it should be not less than %d' %(x.length, x.atLeast)
else:
    print 'Other exception occured'
finally:
    print 'Finally...Done'

上述示例代码中，try中抛出的自定义异常类对象，在except中捕获必须要带两个参数，异常类对象类型（ShortInputException）和形参（x）。

关键字

python中的所有关键字都是小写的，如class、def、import等等。

额外的一点注意

shell中，等待用户输入，Ctrl+d是EOF；Ctrl+c是终止程序。
